CRUD en C++

Proyecto Personal

Este proyecto demuestra una implementación completa de una aplicación CRUD (Crear, Leer, Actualizar, Eliminar) utilizando SQLite y C++. Hace énfasis en conceptos clave de programación como la interacción con bases de datos, manejo de errores y validación de entradas. La aplicación cuenta con una interfaz de consola fácil de usar y con estilo mejorado, permitiendo a los usuarios gestionar datos sin complicaciones. Entre los aspectos destacados del proyecto se incluyen la validación de ID para actualizaciones y eliminaciones seguras, la construcción dinámica de consultas SQL y la creación automatizada de tablas para garantizar la integridad de la base de datos. Este proyecto refleja mi capacidad para desarrollar soluciones robustas, eficientes y escalables, respetando las mejores prácticas en el desarrollo de software.

Herramientas

  • SQLite
  • C++
  • Visual Studio

Password Manager

Proyecto Personal

Password Manager es una aplicación de escritorio desarrollada en Python para gestionar contraseñas de forma segura. Proporciona funciones para agregar, visualizar, actualizar y eliminar contraseñas, almacenándolas en una base de datos encriptada con AES y protegiendo la clave mediante variables de entorno. Cuenta con una interfaz gráfica moderna diseñada con Tkinter y está empaquetada como una aplicación autónoma lista para su distribución.

Herramientas

  • Python
  • SQLite
  • Visual Studio Code
  • Tkinter
  • PyInstaller
  • AES (Advanced Encryption Standard)

EXPERIENCIA

Desarrollador de Software

Vettel Car Wash
2023 - 2023

Como parte de mi formación académica en Ingeniería en Sistemas Computacionales, colaboré con compañeros de equipo en el desarrollo de un proyecto práctico y aplicado en un entorno real con un cliente efectivo. Este proyecto sirvió como la culminación de mis estudios y me proporcionó valiosa experiencia práctica en el uso de HTML, CSS, JavaScript, C#, SQL Server, GitHub, Visual Studio y el .NET Framework.

Objetivo del Proyecto:

El objetivo principal del proyecto fue crear un sistema de gestión de citas para un servicio de lavado de autos. Usando HTML, CSS y JavaScript, diseñamos una interfaz de usuario intuitiva que permite a los clientes agendar citas de manera eficiente. Además, utilizando C# y el .NET Framework, implementamos una funcionalidad robusta en el backend que permite a los empleados del establecimiento gestionar las citas de los clientes, administrar inventarios, administrar perfiles de usuarios y llevar un control de las finanzas asociadas al negocio de lavado de autos.

Responsabilidades y Logros:

• Diseño de Interfaz de Usuario: Creación de una interfaz intuitiva utilizando HTML, CSS y JavaScript para facilitar la programación de citas por parte de los clientes.

• Desarrollo Backend: Implementación de funcionalidades robustas en C# y el .NET Framework para la gestión de citas, inventarios, perfiles de usuarios y finanzas.

• Gestión de Bases de Datos: Uso de SQL Server para la gestión de bases de datos, garantizando la integridad y confiabilidad de los datos.

• Control de Versiones: Uso de GitHub para el control de versiones, manteniendo un flujo de trabajo colaborativo y organizado que facilitó la integración de las contribuciones de código por parte de los miembros del equipo.

• Entorno de Desarrollo: Uso de Visual Studio como el entorno de desarrollo integrado (IDE) principal, aprovechando sus herramientas para la edición de código, depuración y despliegue.

Este proyecto me brindó una experiencia invaluable al aplicar una variedad de tecnologías para resolver problemas del mundo real, ampliando mi comprensión de los conceptos de ingeniería en sistemas computacionales y preparándome para futuros desafíos profesionales.
