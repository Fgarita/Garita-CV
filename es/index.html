Password Manager

Proyecto Personal

Un gestor de contraseñas de escritorio construido en Python. Guarda cada credencial en una base de datos local encriptada con AES, mantiene la clave de encriptación fuera del código usando variables de entorno, y cubre el ciclo completo de CRUD (agregar, ver, actualizar, eliminar) mediante una interfaz gráfica moderna hecha con Tkinter. Empaquetado como ejecutable independiente con PyInstaller, por lo que corre en Windows sin necesitar Python instalado — el tipo de diseño enfocado en seguridad que aprendí en mis certificaciones de Ethical Hacking y Ciberseguridad. ver proyecto
